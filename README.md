# Mikisun Pixel Guardian

Mikisun Pixel Guardian is an original retro handheld-style virtual pet and monster encounter browser game.

This project is currently an experimental prototype built for web play, mobile testing, and gameplay system development.

## Play

Public test version:

https://mikisuno2025-hash.github.io/mikisun-pixel-guardian/

## Current Version

Prototype V33.22

Main features currently include:

* Retro LCD-style handheld interface
* Virtual pet care system
* Feeding, training, cleaning, sleeping, and status check
* Search / patrol encounter system
* Monster battle system
* Evolution route and collection system
* Mobile touch controls
* Mobile shake detection for search
* Google login
* Optional cloud save using Firebase
* Local browser save support
* Multi-language support
* Status debug panel for testing

## Prototype Notice

This game is currently in active development.

Some features, balance values, save behavior, UI layout, and cloud sync logic may change during testing.
Save data may be reset, changed, or become incompatible between prototype versions.

Please treat this as a public test build, not a finished commercial release.

## Save Data

The game may use:

* Browser local storage for local save data
* Firebase Authentication for Google login
* Firebase Firestore for optional cloud save data

Cloud save is intended only for game progress backup and cross-device testing.

No payment, address, or sensitive personal information is required by this game.

## Privacy Notice

When using Google login, the game may receive basic account information provided through Firebase Authentication, such as user ID and email address, for the purpose of identifying the player’s cloud save.

Game progress may be stored under the authenticated user account in Firebase.

This project does not sell user data.
